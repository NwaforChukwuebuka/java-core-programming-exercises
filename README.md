# java-core-programming-exercises
Collection of foundational Java programming exercises covering input/output, arithmetic operations, conditional logic, and basic problem solving. Includes practical tasks like BMI calculation, number analysis, and character operations.

## Folder layout (camelCase)
- **`chapterTwo`** — Chapter 2 textbook exercises
- **`day2/fireDrill1`** & **`day2/fireDrill2`** — Fire drill 1 and 2 (day 2)
- **`dayThree/fireDrill1`** — Day 3 fire drill 1

Run helper: from the repo root, `.\runjava.ps1 YourClassName` (or pass a path like `day2\fireDrill2\TaskEight.java`).
